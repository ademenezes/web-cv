# Ana De Menezes - Interactive CV

A modern, interactive web-based CV showcasing data science and economics expertise.

## Features

- Personalized greeting for Nabila Assaf
- Dark teal theme with modern SVG icons
- Scroll-triggered animations
- Responsive design
- Somalia CCDR experience highlight

## Technologies

- HTML5
- CSS3 (Custom Properties, Animations)
- Vanilla JavaScript
- Modern SVG icons

## Local Development

```bash
python3 -m http.server 8050
```

Then open http://localhost:8050
